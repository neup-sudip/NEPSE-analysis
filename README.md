# NEPSE Stock Market Analysis & Prediction
Short analysis of NEPSE (Nepal Stock Exchange) data from 2000-2021.
It has data of commercial bank, hydro, and microfinance

Tools: Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly, Scikit-learn

Collected and merged historical stock data from NEPSE for Hydropower, Commercial Banks, and Microfinance sectors.

Performed data cleaning: handled missing values, incorrect formats, duplicates, and zero values using domain-specific rules.

Conducted exploratory data analysis (EDA) with trend lines, scatter plots, heatmaps, and pair plots to visualize price and trading patterns.

Analyzed price volatility and identified significant daily changes.

Built machine learning models (Linear Regression & Random Forest Regressor) to predict stock Close Prices with high accuracy using feature engineering on dates and trading metrics.

Provided insights into sector-wise trading behavior and trends in NEPSE.
